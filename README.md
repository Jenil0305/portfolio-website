# Developer Portfolio

An interactive, scrollytelling developer portfolio built with **React**, **Tailwind CSS v4**, **GSAP**, and **Lenis Smooth Scroll**. Designed for frontend engineers who want to showcase advanced web animations and 3D CSS integrations.

## Features
- **Immersive Portal Hero:** A massively scaled SVG typography entry layer that zooms out intuitively on scroll.
- **Horizontal Scroll Toolkit:** Seamless transitions from vertical to horizontal timelines, creating a unique navigation flow.
- **CSS 3D Case Study (MacBook Pro):** A highly performant, CSS-only 3D laptop that opens its lid and dynamically zooms into the contained interface upon scrolling.
- **Scroll-linked Progress Timeline:** A glowing timeline with a scrub-linked position indicator.
- **Magnetic Action Footer:** Fluid color inversion logic coupled with subtle parallax UI forms to drive maximum engagement.

## Tech Stack
- React (bootstrapped with Vite)
- Tailwind CSS v4 (using the `@tailwindcss/vite` plugin system)
- GSAP & ScrollTrigger
- Studio-Freight Lenis
- Lucide React (Icons)

## Local Development

To run this project locally:

1. Clone the repository and navigate into the folder.
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```

## Production Build

To build the project for production (e.g., Vercel, Netlify, Render):
```bash
npm run build
```
This generates the optimized bundle in the `/dist` directory.
